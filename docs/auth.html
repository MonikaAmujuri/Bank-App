Firebase Login

This endpoint allows the user to log in using Firebase ID token and receive a JWT token from the backend.

POST /api/auth/firebase-login 
fetch('http://localhost:5000/api/auth/firebase-login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    idToken: 'eyJhbGciOiJSUzI1NiIsImtpZCI6Ij...'
  })
})
  .then((response) => response.json())
  .then((json) => console.log(json));
👇 Output 
{
  "message": "Login successful",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ...",
  "user": {
    "_id": "699ef3399e1cdb38eec7210d",
    "name": "Admin",
    "email": "admin@bank.com",
    "phone": "+919999999999",
    "role": "admin"
  }
}

Important: The frontend must send a valid Firebase ID token. The backend verifies it and returns your application JWT token.

User Login

This endpoint allows a user to log in using email and password and receive a JWT token from the backend.

POST /api/auth/login 
fetch("http://localhost:5000/api/auth/login", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    email: identifier.toLowerCase().trim(),
    password,
  }),
});
👇 Output 
{
  "message": "Login successful",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ...",
  "user": {
    "id": "69afec0f3aab80c84c6ba09b",
    "name": "User..",
    "email": "user@bank.com",
    "phone": "+917777777777",
    "role": "user"
  }
}

Important: The frontend sends the email in lowercase and trimmed format before login. On success, the backend returns a JWT token and user details.
