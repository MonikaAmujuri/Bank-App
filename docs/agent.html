Agent

This page contains agent-only endpoints for dashboard, users, agents, and loan management.

Agent Dashboard

This endpoint fetches the agent dashboard statistics such as total users, total loans, loan statuses, loan types, and recent loans.

GET /api/agent/dashboard 
fetch("http://localhost:5000/api/agent/dashboard", {
  headers: {
    Authorization: `Bearer ${token}`,
  },
});
👇 Output 
{
  "totalUsers": 2,
  "totalLoans": 3,
  "draftLoans": 1,
  "pendingLoans": 0,
  "approvedLoans": 1,
  "rejectedLoans": 1,
  "homeLoans": 2,
  "personalLoans": 0,
  "educationLoans": 0,
  "vehicleLoans": 0,
  "businessLoans": 0,
  "recentLoans": [
    {
      "_id": "69b5016743bfd732729e80b6",
      "loanId": "LOAN0006"
    },
    ...
  ]
}

Important: This endpoint requires a valid agent JWT token. It returns dashboard counts and recent loan data for the logged-in agent.

Create User

This endpoint allows the agent to create a new user by sending user details and a generated password.

POST /api/users/create-user 
fetch("http://localhost:5000/api/users/create-user", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Authorization: `Bearer ${token}`,
  },
  body: JSON.stringify({
    name,
    email,
    phone,
    panNumber,
    aadharNumber,
    address,
    password: generatedPassword
  }),
});
👇 Output 
{
  "message": "User created successfully",
  "user": {
    "id": "69bcefb6f30336efaee5a07e",
    "userId": "USR0005",
    "name": "User.",
    "email": "user12@example.com",
    "phone": "9876543210",
    "panNumber": "ABCDE1234F",
    "aadharNumber": "123412341234",
    "address": "Guntur, Andhra Pradesh",
    "role": "user"
  },
  "password": "4GUC6FMAFg3x"
}

Important: This endpoint requires a valid agent JWT token. The generated password should be shared securely because it is returned only at user creation time.

Get Agent Users

This endpoint fetches the users created or managed by the logged-in agent. Query parameters can be used to filter the result.

GET /api/agent/users 
fetch(
  `http://localhost:5000/api/agent/users${
  query.toString() ? `?${query.toString()}` : ""
}`,
  {
    headers: {
      Authorization: `Bearer ${token}`,
    },
  }
);
👇 Output 
[
  {
    "_id": "699fed0474e0522950c7fde6",
    "name": "User1",
    "email": "user1@bank.com",

    "isDeleted": false,
    "isActive": true,
    "userId": "USR0001",
    "...": "..."

  ...
]

Important: This endpoint requires a valid agent JWT token. Query parameters can be added dynamically to fetch filtered user data for the logged-in agent.

Get Agent Loans

This endpoint fetches the loan list for the logged-in agent. Query parameters can be used to filter the loans.

GET /api/agent/loans 
fetch(
