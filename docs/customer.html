Customer

This page contains customer-only endpoints for profile, documents, and customer loan access.

Apply Loan

Role: User

This endpoint allows the customer to apply for a loan and submit the loan application data.

POST /api/loans/apply 
fetch("http://localhost:5000/api/loans/apply", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${token}`,
  },
  body: data,
});
👇 Output 
{
  "message": "Loan application submitted successfully",
  "loan": {
    "loanId": "LOAN0009",
    "userId": "USR0001",
    "userObjectId": "699fed0474e0522950c7fde6",
    "agentId": null,
    "createdBy": "699fed0474e0522950c7fde6",
    "createdByRole": "user",
    "loanDetails": {
      "loanType": "Home Loan",
      "amount": 60000
    },
    ...: "..."
  }
}

Important: This endpoint requires a valid user JWT token. The request body is sent as form data, and the backend creates a new loan application for the logged-in customer.

Get My Loan Details

Role: User

This endpoint fetches the complete details of a specific loan belonging to the logged-in customer.

GET /api/user/my-loans/:loanId 
fetch(`http://localhost:5000/api/user/my-loans/${loanId}`, {
  headers: {
    Authorization: `Bearer ${token}`,
  },
});
👇 Output 
{
  "_id": "699feebf74e0522950c7fdfb",
  "loanId": "LOAN0001",
  "userObjectId": "699fed0474e0522950c7fde6",
  "agentId": "699fd764dd531d0dcaf9ab66",
  "status": "rejected",
  "remarks": "Loan details are not valid",
  "loanDetails": {
    "amount": "600000",
    "tenure": 24,
    "loanType": "Home Loan",
    "interestRate": "5",
    "tenureMonths": "6",
    "purpose": "for home"
  },
  ...: "..."

Important: This endpoint requires a valid user JWT token. Replace :loanId with the actual loan ID to fetch that loan’s full details. It returns only loans that belong to the logged-in customer.

Update My Profile

Role: User

This endpoint allows the customer to update profile details using JSON form data.

PUT /api/users/me 
fetch("http://localhost:5000/api/users/me", {
  method: "PUT",
  headers: {

    Authorization: `Bearer ${token}`,
  },
  body: JSON.stringify(formData),
});
