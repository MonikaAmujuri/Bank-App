Admin

This page contains admin-only endpoints for dashboard, users, agents, and loan management.

Admin Dashboard

This endpoint fetches admin dashboard statistics like users, agents, total loans, loan statuses, and loan type counts.

GET /api/admin/dashboard 
const fetchStats = async () => {
  try {
    const res = await fetch("http://localhost:5000/api/admin/dashboard", {
      headers: {
        Authorization: `Bearer ${token}`,
      },
    });

    const data = await res.json();
  } catch (error) {
    console.log(error);
  }
};
👇 Output 
{
  "users": 5,
  "agents": 2,
  "loans": 7,
  "pendingLoans": 0,
  "draftLoans": 3,
  "approvedLoans": 3,
  "rejectedLoans": 1,
  "homeLoans": 3,
  "personalLoans": 1,
  "educationLoans": 0,
  "businessLoans": 1,
  "vehicleLoans": 0
}

Important: This endpoint requires an admin JWT token in the Authorization header. It returns overall dashboard statistics for the admin panel.

Create Agent

This endpoint allows the admin to create a new agent by sending basic agent details. The backend returns the created agent data and generated password.

POST /api/admin/create-agent 
fetch("http://localhost:5000/api/admin/create-agent", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Authorization: `Bearer ${token}`,
  },
  body: JSON.stringify({ name, email, phone, location, pincode }),
});
